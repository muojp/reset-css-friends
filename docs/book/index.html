はじめに

本書はReset CSSについて、Reset CSSとは何か、なぜ・どうやって使うかを解説していく本です。ブラウザが既定で読み込むユーザーエージェントスタイルシートを元に、Reset CSSでなぜそのプロパティや値が定義されているか、数あるReset CSSでどういった定義がされているのか解説していきます。
